Lista de Tarefas


Instruçoes:

  • Digite uma tarefa no campo abaixo e clique em Enter ou no botao 'Adicionar'.
  • Mova as tarefas com o mouse para reorganizá-las na lista.
  • Utilize a checkbox para marcar as tarefas após completá-las.
  • Clique duas vezes na tarefa para marcá-la como algo 'irrelevante'.
  • Clique no "X" para remover a tarefa da lista.
